# 题单测试

## 前置条件

- dev 环境运行中
- 预置 3 个题单：
  - 题单1「新手入门题单」（公开，admin 创建）
  - 题单2「动态规划专题」（公开，teacher1 创建）
  - 题单3「竞赛选拔题集」（非公开，teacher2 创建，student1/student2 有 READ 权限）

## 测试用例

### PS1 — 浏览题单列表（游客）

- **步骤**：
  1. 不登录，访问 `/problemset`
- **预期**：
  - 显示公开题单：「新手入门题单」、「动态规划专题」
  - 不显示非公开题单「竞赛选拔题集」

### PS2 — 查看题单详情

- **角色**：student1
- **步骤**：
  1. 登录 student1
  2. 点击「新手入门题单」
- **预期**：
  - 显示 4 道题目：P0001(a+b)、P0002(数组求和)、P1001(区间求和)、P1004(两数之和)
  - 按排序顺序排列

### PS3 — 有权限查看私有题单

- **角色**：student1（有题单3的 READ 权限）
- **步骤**：
  1. 登录 student1
  2. 在题单列表中查找「竞赛选拔题集」或直接访问 `/problemset/3`
- **预期**：
  - 可以查看题单详情
  - 显示 3 道题目：P1007(逆序对)、P1009(迷宫寻路)、P1010(岛屿计数)

### PS4 — 无权限查看私有题单

- **角色**：student3（无题单3的权限）
- **步骤**：
  1. 登录 student3（孙同学）
  2. 尝试访问 `/problemset/3`
- **预期**：
  - 不可见或提示无权限
  - 题单列表中不显示该题单

### PS5 — 创建题单

- **角色**：teacher1
- **步骤**：
  1. 登录 teacher1
  2. 访问 `/problemset/create`
  3. 填写标题「搜索专题」，描述「BFS/DFS 练习」
  4. 设置为公开
  5. 创建
- **预期**：
  - 题单创建成功
  - 在题单列表中可见

### PS6 — 编辑题单（添加/修改题目）

- **角色**：teacher1
- **步骤**：
  1. 进入「搜索专题」的编辑页面（`/problemset/:id/edit`）
  2. 添加题目 P1009（迷宫寻路）和 P1010（岛屿计数）
  3. 保存
- **预期**：
  - 题目列表更新
  - 题单详情显示 2 道题

### PS7 — 授权用户访问私有题单

- **角色**：teacher2（题单3 的创建者）
- **步骤**：
  1. 登录 teacher2
  2. 给 student3 授予题单3 的 READ 权限
- **预期**：
  - 授权成功
  - student3 登录后可以查看题单3

### PS8 — 删除题单

- **角色**：teacher1
- **步骤**：
  1. 进入「搜索专题」详情
  2. 点击删除
- **预期**：
  - 题单被软删除
  - 题单列表中不再显示
