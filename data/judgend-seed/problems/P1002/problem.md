## 题目描述

给定一个长度为 $n$ 的整数序列 $a_1, a_2, \dots, a_n$，请找到一个**连续的非空子段** $a_l, a_{l+1}, \dots, a_r$，使得这个子段的和尽可能大。

输出这个最大的子段和。

## 输入格式

第一行一个正整数 $n$ ($1 \le n \le 2 \times 10^5$)。

第二行 $n$ 个整数 $a_1, a_2, \dots, a_n$ ($|a_i| \le 10^9$)。

## 输出格式

一行一个整数，表示最大子段和。

## 样例

### 样例 1

#### 输入

```
6
-2 1 -3 4 -1 2
```

#### 输出

```
5
```

选取子段 $[4, -1, 2]$，和为 $5$。

### 样例 2

#### 输入

```
3
-1 -2 -3
```

#### 输出

```
-1
```

所有元素为负，选最大的 $-1$。

## 提示

子段不能为空，至少包含一个元素。当所有元素均为负数时，最大子段和就是最大的那个元素。
