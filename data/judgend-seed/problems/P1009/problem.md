## 题目描述

给定一个 $n \times m$ 的网格迷宫，其中 `.` 表示空地（可通行），`#` 表示障碍（不可通行）。

从左上角 $(1,1)$ 出发，每步可以向**上、下、左、右**移动一格（不能走出网格边界，也不能走到障碍上），目标是到达右下角 $(n,m)$。

请求出从起点到终点的最少步数。如果无法到达，输出 $-1$。

## 输入格式

第一行两个正整数 $n, m$ ($1 \le n, m \le 1000$)。

接下来 $n$ 行，每行一个长度为 $m$ 的字符串，仅包含 `.` 和 `#`。

## 输出格式

一个整数，表示最少步数。如果无法到达，输出 $-1$。

## 样例

### 样例 1

#### 输入

```
3 3
...
.#.
...
```

#### 输出

```
4
```

### 样例 2

#### 输入

```
3 3
..#
.#.
#..
```

#### 输出

```
-1
```

右下角被障碍包围，无法到达。

### 样例 3

#### 输入

```
1 1
.
```

#### 输出

```
0
```

起点即终点。
