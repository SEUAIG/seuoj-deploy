## 题目描述

给定一个只包含 `(` 和 `)` 的字符串 $S$，请判断它是否是一个**合法的括号序列**。

- 如果是合法的括号序列，输出它的**最大嵌套深度**。
- 如果不是合法的括号序列，输出 $-1$。

例如，`(()())` 是合法的，最大嵌套深度为 $2$；而 `(()` 不合法。

## 输入格式

一行一个字符串 $S$，仅包含 `(` 和 `)` ($1 \le |S| \le 2 \times 10^5$)。

## 输出格式

一个整数。如果 $S$ 是合法括号序列，输出最大嵌套深度；否则输出 $-1$。

## 样例

### 样例 1

#### 输入

```
(()())
```

#### 输出

```
2
```

### 样例 2

#### 输入

```
((()))
```

#### 输出

```
3
```

### 样例 3

#### 输入

```
)(
```

#### 输出

```
-1
```

### 样例 4

#### 输入

```
(
```

#### 输出

```
-1
```
