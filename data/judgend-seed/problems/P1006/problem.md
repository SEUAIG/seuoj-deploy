## 题目描述

在一片果园里有 $n$ 堆果子，第 $i$ 堆有 $a_i$ 个果子。

每次可以选择**两堆**果子合并成一堆，合并的体力消耗等于这两堆果子的数量之和。

经过 $n-1$ 次合并后，所有果子会合为一堆。请问：最少需要消耗多少体力？

## 输入格式

第一行一个正整数 $n$ ($2 \le n \le 10^5$)。

第二行 $n$ 个正整数 $a_1, a_2, \dots, a_n$ ($1 \le a_i \le 10^5$)。

## 输出格式

一个整数，表示最小体力消耗。

## 样例

### 样例 1

#### 输入

```
3
1 2 9
```

#### 输出

```
15
```

先合并 $1$ 和 $2$，消耗 $3$；再合并 $3$ 和 $9$，消耗 $12$。总计 $15$。

### 样例 2

#### 输入

```
4
1 1 1 1
```

#### 输出

```
8
```

## 提示

每次选择最小的两堆合并，贪心策略的正确性可以类比 Huffman 编码。
