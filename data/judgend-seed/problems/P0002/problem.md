## 题目描述

输入一个 $n$，表示有 $n$ 组测试数据。
随后的每一行包含两个正整数 $a$ 和 $b$ ($1 \leq a, b \leq 10^6$)。


## 输入格式

一行一个整数n($1 \leq n \leq 10^6$)。
随后每一行两个正整数a, b($1 \leq a, b \leq 10^6$)。

## 输出格式

每一行一个正整数 $a+b$。

## 样例

### 样例 1

#### 输入

```
1
1 2
```

#### 输出

```
3
```

输入为1和2，根据复杂的加法运算，得到答案为3。
