## 题目描述

给定一个**升序排列**的整数数组 $a_1, a_2, \dots, a_n$ 和一个目标值 $\text{target}$，请找到两个不同的下标 $i, j$ ($1 \le i < j \le n$)，使得 $a_i + a_j = \text{target}$。

保证至少存在一组解。如果有多组解，输出任意一组即可。

## 输入格式

第一行两个整数 $n$ 和 $\text{target}$ ($2 \le n \le 2 \times 10^5$, $|\text{target}| \le 2 \times 10^9$)。

第二行 $n$ 个整数 $a_1, a_2, \dots, a_n$ ($|a_i| \le 10^9$)，保证 $a_1 \le a_2 \le \dots \le a_n$。

## 输出格式

一行两个整数 $i, j$ ($i < j$)，表示满足条件的一对下标。

## 样例

### 样例 1

#### 输入

```
5 9
1 2 4 5 7
```

#### 输出

```
2 5
```

$a_2 + a_5 = 2 + 7 = 9$

### 样例 2

#### 输入

```
3 6
1 3 5
```

#### 输出

```
1 3
```

$a_1 + a_3 = 1 + 5 = 6$

## 提示

尝试使用双指针从两端向中间逼近。
