## 题目描述

给定一个长度为 $n$ 的整数序列 $a_1, a_2, \dots, a_n$。

如果存在 $i < j$ 且 $a_i > a_j$，则称 $(i, j)$ 为一个**逆序对**。

请计算序列中逆序对的总数。

## 输入格式

第一行一个正整数 $n$ ($1 \le n \le 2 \times 10^5$)。

第二行 $n$ 个整数 $a_1, a_2, \dots, a_n$ ($1 \le a_i \le 10^9$)。

## 输出格式

一个整数，表示逆序对的数量。

## 样例

### 样例 1

#### 输入

```
5
2 3 8 6 1
```

#### 输出

```
5
```

逆序对为 $(1,5),(2,5),(3,4),(3,5),(4,5)$。

### 样例 2

#### 输入

```
3
1 2 3
```

#### 输出

```
0
```

已经有序，没有逆序对。

## 提示

暴力枚举是 $O(n^2)$ 的，试试归并排序的思想。
