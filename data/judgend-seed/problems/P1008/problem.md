## 题目描述

给定一个由小写字母组成的字符串 $S$，请重新排列其中的字符，使其成为一个**回文串**。

- 如果可以，输出任意一个合法的回文串。
- 如果不可以，输出 `impossible`。

回文串是指正着读和反着读都一样的字符串，例如 `abcba` 和 `abba`。

## 输入格式

一行一个字符串 $S$，仅包含小写字母 ($1 \le |S| \le 10^5$)。

## 输出格式

一行，一个回文串或 `impossible`。

## 样例

### 样例 1

#### 输入

```
aabb
```

#### 输出

```
abba
```

其他合法答案如 `baab` 也可以。

### 样例 2

#### 输入

```
abc
```

#### 输出

```
impossible
```

三个不同字符，无法构成回文。

### 样例 3

#### 输入

```
aaabb
```

#### 输出

```
ababa
```

## 提示

想想什么条件下才能构成回文串？与字符出现次数的奇偶性有关。
