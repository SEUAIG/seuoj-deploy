## 题目描述

有 $n$ 条绳子，长度分别为 $L_1, L_2, \dots, L_n$（均为正整数）。现在需要从这些绳子中切出 $k$ 条**等长**的绳子段，每条绳子段的长度必须为正整数。

问切出的绳子段最长能有多长？如果无法切出 $k$ 条绳子段，输出 $0$。

**注意**：每条绳子只能从中切割，不能将不同的绳子拼接在一起。

## 输入格式

第一行两个正整数 $n, k$ ($1 \le n \le 10^5$, $1 \le k \le 10^9$)。

第二行 $n$ 个正整数 $L_1, L_2, \dots, L_n$ ($1 \le L_i \le 10^9$)。

## 输出格式

一个整数，表示切出的绳子段的最大长度。如果无法切出 $k$ 条，输出 $0$。

## 样例

### 样例 1

#### 输入

```
4 11
802 743 457 539
```

#### 输出

```
200
```

每段长 $200$，第一条切 $4$ 段，第二条 $3$ 段，第三条 $2$ 段，第四条 $2$ 段，共 $11$ 段。

### 样例 2

#### 输入

```
2 5
1 1
```

#### 输出

```
0
```

两条长度为 $1$ 的绳子最多切 $2$ 条长度为 $1$ 的段，无法得到 $5$ 条。

## 提示

尝试二分答案：如果每段长度为 $x$，能否切出至少 $k$ 条？
