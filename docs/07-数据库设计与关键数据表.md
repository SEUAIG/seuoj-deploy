# 数据库设计与关键数据表

## 1. 概述

| 项目 | 值 |
|------|---|
| 数据库 | MySQL 8.0 |
| 数据库名 | seuoj |
| 字符集 | utf8mb4 (unicode_ci) |
| ORM | MyBatis Plus 3.5.9 |
| 初始化脚本 | `data/init/init.sql` + `data/init/seed.sql` |
| 表数量 | 22 张 |

### 通用设计约定

- **主键**: 自增 `id` (BIGINT)
- **创建者**: `created_by_user_id` (仅信息记录，权限判定走 resource_permission)
- **软删除**: `is_del` TINYINT (0=存在, 1=已删除)
- **时间戳**: `created_at` (创建时自动填充), `updated_at` (更新时自动维护)
- **唯一约束**: Generated Column `active_key`，仅对未删除记录生效
- **外键**: 定义了外键约束，ON DELETE RESTRICT

### 3.9 权限模块

#### resource_permission（统一资源权限）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 内部主键 |
| resource_type | VARCHAR(32) | NOT NULL | 资源类型：PROBLEM/PROBLEM_SET/CLASS/CONTEST |
| resource_id | BIGINT | NOT NULL | 资源内部 ID |
| user_id | BIGINT | NOT NULL | 被授权用户 ID |
| permission | VARCHAR(16) | NOT NULL | 权限类型：READ/WRITE（WRITE 蕴含 READ） |
| granted_by | BIGINT | NULL | 授权者用户 ID，NULL 表示系统自动授权 |
| created_at | DATETIME | 创建时间 |
| is_del | TINYINT | 软删除 |

**索引**: UNIQUE(active_key), INDEX(user_id, resource_type), INDEX(resource_type, resource_id)

**设计原则**：
- WRITE 蕴含 READ
- 拥有 WRITE 权限的用户可以对同一资源授予他人任意权限
- 资源创建时自动插入一条 WRITE 记录给创建者
- `created_by_user_id` 仅作为信息字段，不参与权限判定

**替代关系**：
| 已删除的表 | 在 resource_permission 中的表达 |
|-----------|-------------------------------|
| `problem_set_invited_member_rel` | `(PROBLEM_SET, ps_id, user_id, READ/WRITE)` |
| `contest_manager_rel` | `(CONTEST, contest_id, user_id, WRITE)` |
| `user_problem_set_rel` | `(PROBLEM_SET, ps_id, user_id, READ)` |
| `class_problem_set_rel` | 由 `assignment` 表替代 |

## 5. 数据完整性设计

### 外键约束

所有关联表均定义了外键，使用 `ON DELETE RESTRICT` 防止误删。

### 软删除

- 所有表使用 `is_del` 字段实现软删除
- MyBatis Plus 全局配置 `logic-delete-field: is_del`
- Generated Column `active_*` 确保唯一约束仅对未删除记录生效

### Generated Column 示例

```sql
active_key VARCHAR(128) GENERATED ALWAYS AS (
    CASE WHEN is_del = 0 THEN CONCAT(class_id, '#', contest_id) ELSE NULL END
) STORED
```

## 6. 索引策略

| 表 | 索引 | 类型 | 说明 |
|----|----|------|------|
| problem | pid | UNIQUE | 唯一题号 |
| problem | title | FULLTEXT (ngram) | 全文搜索 |
| submission | submission_no | UNIQUE | 提交编号 |
| submission | user_id | INDEX | 用户提交查询 |
| submission | problem_id | INDEX | 题目提交查询 |
| resource_permission | active_key | UNIQUE | 防重复授权 |
| resource_permission | (user_id, resource_type) | INDEX | 用户权限查询 |
| resource_permission | (resource_type, resource_id) | INDEX | 资源权限查询 |
| 所有关联表 | active_key | UNIQUE | 防重复关联 |
