# 快速开始

## 注册账号

1. 访问系统首页，点击右上角「注册」按钮
2. 跳转到注册页面（`/signup`），填写以下信息：
   - **用户名**：登录名，全局唯一
   - **邮箱**：用于接收验证码和找回密码
   - **密码**：设置登录密码
3. 点击「发送验证码」，系统会向填写的邮箱发送 6 位数字验证码
4. 输入验证码后点击「注册」
5. 注册成功后自动登录并跳转到首页

## 登录

1. 访问系统首页，点击右上角「登录」按钮
2. 跳转到登录页面（`/login`）
3. 输入用户名（或邮箱）和密码
4. 点击「登录」，成功后跳转到首页

## 忘记密码

1. 在登录页面点击「忘记密码」链接
2. 跳转到密码重置页面（`/forget`）
3. 输入注册时使用的邮箱
4. 点击「发送验证码」
5. 输入验证码和新密码
6. 点击「重置密码」，完成后可用新密码登录

## 界面概览

登录后，顶部导航栏包含以下入口：

| 入口 | 路径 | 说明 |
|------|------|------|
| 班级 | `/class` | 查看和加入班级 |
| 比赛 | `/contest` | 浏览和参加比赛 |
| 练习 > 题库 | `/problemsLibrary` | 浏览和搜索题目 |
| 练习 > 评测 | `/evaluation` | 查看提交记录 |
| 练习 > 题单 | `/problemset` | 浏览题单 |
| 更多 > 帮助 | `/help` | 帮助页面 |
| 更多 > 管理 | `/admin/users` | 管理员专用，仅管理员可见 |

点击右上角头像可进入个人中心。
