# 班级

## 浏览班级列表

访问「班级」（`/class`），可以看到班级列表。包含公开班级和你已加入的班级。

列表显示班级名称和简要描述。

## 加入班级

1. 在班级列表中点击某个公开班级
2. 进入班级详情页（`/class/:id`）
3. 点击「加入班级」按钮
4. 加入成功后可看到班级的完整内容

> 非公开班级需要教师将你添加为成员才能访问。

## 班级详情

进入班级详情页后，可查看以下内容：

### 班级简介

班级的 Markdown 富文本介绍，包含课程说明、考核方式等信息。

### 公告

班级公告按时间排列，置顶公告会显示在最上方。公告由教师发布，用于通知课程安排变更、作业补充说明等。

### 作业

班级中已发布的作业列表。点击作业可查看详情和提交代码，参见 [作业](05-assignments.md)。

### 关联比赛

教师可能将比赛关联到班级，在班级详情中可以看到关联比赛的入口。

### 关联题单

教师可能将题单关联到班级，在班级详情中可以看到关联题单的入口。

### 成员列表

查看班级中所有成员。
