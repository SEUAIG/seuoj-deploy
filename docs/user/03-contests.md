# 比赛

## 浏览比赛列表

访问「比赛」（`/contest`），可以看到比赛列表。比赛有三种状态：

| 状态 | 说明 |
|------|------|
| 未开始 | 比赛尚未到开始时间 |
| 进行中 | 当前时间在比赛的开始和结束时间之间 |
| 已结束 | 比赛已过结束时间 |

公开比赛所有人可见；非公开比赛仅有权限的用户可见（例如通过班级关联的比赛）。

## 报名与取消报名

1. 在比赛列表中点击比赛标题，进入比赛详情页（`/contest/:id`）
2. 点击「报名」按钮
3. 报名成功后可在比赛期间提交代码
4. 如需取消报名，在比赛详情页点击「取消报名」

## 比赛中做题

1. 进入比赛详情页（`/contest/:id`）
2. 可以看到比赛的题目列表（以字母顺序标记：A、B、C...）
3. 点击题目进入比赛题目详情页（`/contest/:id/:pid`）
4. 在代码编辑器中编写代码并提交
5. 提交后可在比赛提交列表（`/contest/:id/submissions`）中查看自己的提交记录

## 赛制说明

SEUOJ 支持四种赛制：

### ACM 赛制
- 每道题只有「通过（AC）」和「未通过」两种状态
- 排名按通过题目数从多到少排列
- 相同通过数按总罚时从少到多排列
- 罚时 = 各通过题目的首次通过时间 + 错误提交次数 × 20 分钟

### IOI 赛制
- 每道题支持部分得分（0~100 分）
- 取每道题所有提交中的最高分
- 排名按总分从高到低排列

### NOI 赛制
- 每道题支持部分得分（0~100 分）
- 取每道题的**最后一次提交**的得分（而非最高分）
- 排名按总分从高到低排列

### CUSTOM 赛制
- 自定义计分规则
- 由比赛创建者通过脚本定义计分逻辑

## 排行榜

在比赛详情页可查看排行榜（Standings），显示所有参赛者的排名、各题得分/状态、总分/罚时。

> 部分比赛可能在比赛期间隐藏实时统计（`hide_statistics`），此时排行榜在比赛结束后才可见。

## 查看比赛提交记录

在比赛提交列表页（`/contest/:id/submissions`），可以查看自己的提交记录，包括提交时间、判题结果、得分等。

点击某条提交记录可查看详细信息（`/contest/:id/submission/:submission_no`）。
