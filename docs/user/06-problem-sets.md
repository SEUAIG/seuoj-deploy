# 题单

题单是按主题组织的题目集合，方便进行专项练习。

## 浏览题单列表

访问「练习 > 题单」（`/problemset`），可以看到题单列表。

你可以看到：
- **公开题单**：所有人可见
- **私有题单**：仅被授权的用户可见（教师通过权限管理授予 READ 权限）

列表显示题单标题、描述和创建者。

## 查看题单详情

点击题单标题进入题单详情页（`/problemset/:id`），可查看：

- 题单标题和描述
- 题目列表（按排序顺序排列）
- 每道题的通过状态

## 刷题练习

题单中的题目按顺序排列。点击题目进入题目详情页，提交代码后返回题单可查看完成进度。
