# 教师指南

本章面向 TEACHER 及以上角色，介绍创建和管理教学资源的操作流程。

## 题目管理

### 创建题目

1. 访问题库页面（`/problemsLibrary`）
2. 点击「创建题目」按钮，进入创建页面（`/problemsLibrary/create`）
3. 填写以下信息：
   - **题目编号（PID）**：系统建议下一个可用编号，也可自定义
   - **题目标题**
   - **是否公开**：公开后所有用户可见
4. 点击「创建」保存

### 编辑题面

1. 在题目详情页点击「编辑」，进入编辑页面（`/problemsLibrary/:id/edit`）
2. 使用 Markdown 编辑器编写题面内容，支持：
   - KaTeX 数学公式
   - 代码块
   - 表格
   - 图片
3. 编写输入/输出格式说明和样例
4. 保存修改

### 管理测试数据

1. 在题目详情页点击「测试数据」，进入管理页面（`/problemsLibrary/:id/testfile`）
2. 上传测试数据文件（通常为 `.in` 和 `.out` 文件对）
3. 可以查看已上传的文件列表和文件树
4. 可以删除不需要的测试数据文件

### 配置判题参数

1. 在题目详情页点击「判题配置」，进入配置页面（`/problemsLibrary/:id/judgeConfig`）
2. 设置以下参数：
   - 时间限制（ms）
   - 内存限制（MB）
   - 判题模式（标准比较 / 特殊判题 / 交互判题）
3. 保存配置

### 编辑标签

为题目添加或修改标签，帮助学生按分类筛选题目。

### 删除题目

在题目详情页可删除自己创建的题目（软删除）。

## 比赛管理

### 创建比赛

1. 访问比赛页面（`/contest`），点击「创建比赛」
2. 进入创建页面（`/contest/create`），填写：
   - **标题和副标题**
   - **描述**：Markdown 格式
   - **开始时间和结束时间**
   - **赛制**：ACM / IOI / NOI / CUSTOM
   - **是否公开**
   - **是否隐藏实时统计**
3. 创建后进入编辑页面添加题目

### 编辑比赛

在比赛详情页点击「编辑」（`/contest/:id/edit`），可修改比赛信息和题目列表。

#### 添加题目到比赛

在比赛编辑页中，通过题目编号（PID）搜索并添加题目，设置题目排序。

### 查看比赛提交和排行榜

可以查看所有参赛者的提交记录和排行榜。

## 班级管理

### 创建班级

1. 访问班级页面（`/class`），点击「创建班级」
2. 填写班级名称、描述、Markdown 介绍
3. 设置是否公开
4. 创建成功

### 编辑班级

在班级详情页点击「编辑」，可修改名称、描述、介绍、公开状态。

### 管理成员

#### 单独添加成员

在成员管理中通过用户 ID 添加成员。

#### 批量导入学生

1. 在班级详情页点击「批量导入」
2. 下载 Excel 模板或按格式准备文件（参考 `data/test/batch-import/simport-success.xlsx`）
3. 上传 Excel 文件
4. 系统处理：
   - 用户不存在 → 自动创建账号并加入班级
   - 用户已存在且未加入 → 加入班级
   - 用户已加入 → 跳过
5. 查看导入结果报告

#### 移除成员

在成员列表中选择成员并移除。

### 发布公告

1. 在班级详情的公告区域点击「发布公告」
2. 填写公告标题和 Markdown 内容
3. 可设置是否置顶
4. 发布后所有班级成员可见

### 关联比赛

将已有比赛关联到班级，班级成员可在班级详情中直接进入比赛。

### 关联题单

将已有题单关联到班级，班级成员可在班级详情中直接进入题单。

### 班级总览

访问班级总览（`/class/:id` 的概览标签页），可查看：
- 所有学生的作业完成情况汇总
- 各学生的做题数量和通过率

## 作业管理

### 创建作业

1. 在班级详情页进入作业管理
2. 点击「创建作业」
3. 填写：
   - **标题和描述**
   - **Markdown 介绍**（可附带附件）
   - **截止时间**（可选，NULL 表示无截止时间）
   - **可见时间段**（visible_from ~ visible_to）
4. 创建后默认状态为 DRAFT（草稿）

### 管理作业题目

1. 进入作业编辑页
2. 通过题目编号添加题目
3. 为每道题设置：
   - **排序序号**：决定题目显示顺序
   - **权重**：影响该题在总分中的占比，默认为 1

也支持通过 Excel 批量导入题目列表。

### 发布/关闭作业

| 操作 | 效果 |
|------|------|
| 发布（DRAFT → PUBLISHED） | 学生可见并可提交 |
| 关闭（PUBLISHED → CLOSED） | 学生仍可查看但不能提交 |

### 作业公告

在作业下发布公告，用于补充说明、提示等。

### 作业总览

查看作业的学生完成统计：
- 每个学生每道题的提交数和最高得分
- 班级整体完成率

## 题单管理

### 创建题单

1. 访问题单页面（`/problemset`），点击「创建题单」（`/problemset/create`）
2. 填写标题和描述
3. 设置是否公开
4. 添加题目并设置排序

### 编辑题单

在题单详情页点击「编辑」（`/problemset/:id/edit`），可修改标题、描述、题目列表。

## 权限管理

资源权限系统支持对以下资源类型进行细粒度权限控制：

| 资源类型 | 说明 |
|----------|------|
| PROBLEM | 题目 |
| PROBLEM_SET | 题单 |
| CLASS | 班级 |
| CONTEST | 比赛 |

权限类型：
- **READ**：可查看资源
- **WRITE**：可编辑资源（蕴含 READ）

### 授予权限

可以将某个资源的 READ 或 WRITE 权限授予指定用户。例如将私有题单的 READ 权限授予某个学生。

### 撤销权限

可以撤销之前授予的权限。

> 创建资源时，系统会自动为创建者授予 WRITE 权限。
