# 个人中心

## 查看个人信息

点击右上角头像进入个人中心（`/personal`），可查看：

- **用户名**：登录名
- **昵称**：显示名称
- **邮箱**：注册邮箱
- **角色**：当前角色（学生/教师/管理员/超级管理员）

## 修改个人资料

在个人中心可修改昵称等基本信息。

## 提交热力图

个人中心展示过去一段时间的提交热力图（类似 GitHub contribution graph），直观显示每日提交频率。

## 修改密码

1. 登录后访问修改密码功能
2. 输入当前密码
3. 输入新密码并确认
4. 点击「修改密码」

修改成功后需要使用新密码重新登录。
