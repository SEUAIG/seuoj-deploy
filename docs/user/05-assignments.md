# 作业

作业隶属于班级，由教师创建和管理。

## 查看作业列表

在班级详情页中查看该班级的作业列表。

作业有三种状态：
| 状态 | 说明 |
|------|------|
| PUBLISHED | 已发布，学生可查看和提交 |
| CLOSED | 已关闭，学生可查看但不能提交 |
| DRAFT | 草稿，学生不可见 |

## 查看作业详情

点击作业标题进入作业详情页（`/class/:classId/assignment/:assignmentId`），可查看：

- **作业描述**：题目说明和要求
- **作业介绍**：Markdown 富文本（可能包含附件）
- **截止时间**：`deadline`，过期后不再接受提交
- **可见时间段**：`visible_from` ~ `visible_to`
- **题目列表**：
  - 题目名称和编号
  - 排序顺序
  - 题目权重（影响该题在作业总分中的占比）
  - 你的提交状态

## 提交作业代码

1. 在作业详情页点击某道题目
2. 进入题目详情页，编写代码并提交
3. 提交会自动关联到该作业（`assignment_id`）
4. 返回作业详情页可查看各题提交状态

## 作业公告

教师可能在某个作业下发布公告（例如补充说明、提示等），在作业详情页中查看。

## 注意事项

- 请关注截止时间，超过截止时间或作业状态变为 CLOSED 后将无法提交
- 每道题可以多次提交，系统记录所有提交的判题结果
