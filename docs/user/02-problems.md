# 题库

## 浏览题目列表

访问「练习 > 题库」（`/problemsLibrary`），可以看到所有公开题目的列表。

列表显示以下信息：
- **题目编号**（PID）：如 P0001
- **题目标题**
- **通过率**：总提交数和通过数

### 搜索题目

在搜索框中输入关键词（题目标题），支持模糊搜索。例如输入「区间」可找到「区间求和」。

### 按标签筛选

题目可以通过标签进行筛选，标签分为以下几组：

| 标签组 | 示例标签 |
|--------|---------|
| 基础算法 | 贪心、动态规划、前缀和、栈、双指针、二分、分治、字符串、搜索 |
| 高级算法 | 图论 |
| 赛事来源 | NOI、NOIP、ICPC、IOI、校内赛 |
| 年份 | 2000、2001、2002、2003 |
| 特殊 | 交互题、提交答案、O2优化 |

## 查看题目详情

点击题目标题进入题目详情页（`/problemsLibrary/:id`），可查看：
- 题目描述（支持 Markdown 和 KaTeX 数学公式）
- 输入/输出格式说明
- 样例输入输出
- 时间限制和内存限制

## 提交代码

1. 在题目详情页下方，选择编程语言：
   - C
   - C++
   - Java
   - Python
   - Go
   - Node.js
2. 在代码编辑器（Monaco Editor）中编写代码
3. 点击「提交」按钮
4. 系统会跳转到提交详情页，等待判题结果

> 提交代码需要登录。未登录用户点击「提交」会跳转到登录页面。
