# SEUOJ 用户手册

SEUOJ 是东南大学在线评测系统（Online Judge），面向高校师生提供算法题目练习、编程竞赛、课程作业管理等功能。

## 系统角色

| 角色 | 说明 |
|------|------|
| 游客 | 未登录用户，可浏览公开题库、比赛列表、题单等 |
| 学生（STUDENT） | 注册用户默认角色，可做题、参赛、加入班级、提交作业 |
| 教师（TEACHER） | 可创建和管理题目、比赛、班级、作业、题单 |
| 管理员（ADMIN） | 可管理用户角色（学生/教师），批量导入用户 |
| 超级管理员（SUPER_ADMIN） | 最高权限，可授予管理员角色 |
