## 4. 状态管理

### Redux Slices (8 个，全部通过 redux-persist 持久化到 localStorage)

| Slice | 持久化字段 | 核心状态 |
|-------|-----------|---------|
| auth | user, isAuthenticated, jwt | 用户信息、登录态、JWT Token、角色 |
| code | language, fontsize, codeFileObjectArray | 编辑器语言、字号、代码缓存（按 pid） |
| contestCode | language, fontsize, codeFileObjectArray | 竞赛独立的代码状态（按 contest_id+pid） |
| problemList | — | 分页参数、搜索关键词、标签筛选 |
| contestList | — | 竞赛列表筛选：状态、关键词、时间、赛制 |
| submissionList | — | 提交列表分页参数 |
| tags | — | 标签数据 |
| verification | verificationID, expireAt, nextSendAt | 验证码 ID、发送状态、过期时间 |

**authSlice 特性**：
- 使用 `createAsyncThunk` 处理 login/signup
- 重持久化时检查 JWT 过期，自动清除过期认证状态
- 角色迁移：旧 `"user"` → `"student"`
- 登录成功触发 `onLoginSuccess` 回调，清空所有 React Query 缓存

### TanStack React Query

- staleTime: 60 秒
- retry: 1 次
- 不自动 refetchOnWindowFocus
- 全局错误处理 → Sonner Toast 提示
- 主要 Query Keys：`getTags`, `getContestDetail`, `getContestList`, `getSubmission` 等

## 5. API 服务层

### 双 API 客户端

1. **Axios 实例** (`/services/api/axios.ts`)
   - 基础 URL：环境变量 `VITE_API_BASE_URL`
   - 10 秒超时
   - 请求拦截器：自动注入 JWT Bearer Token
   - 响应拦截器：401 时重置登录态，提取错误消息

2. **Raw fetch** (`/services/ai/agentChat.ts`)
   - 用于 AI/RAG Agent 端点 (`/agent/api/rag/...`)
   - 5 分钟超时（rag_answer 请求）
   - 手动构建 auth headers

### API 模块分布

| 模块 | 路径前缀 | 主要接口 |
|------|---------|---------|
| 认证 | `/api/auth` | login, register, sendCode, resetPassword |

| 竞赛 | `/api/contest` | page, detail, problems, submission, standings, register |
| 题单 | `/api/problem_set` | page, detail, create, update, replaceProblems |

## 10. 样式系统

- **框架**: Tailwind CSS 3.4 + `@tailwindcss/vite`
- **暗色模式**: CSS class 策略（`html.dark`），通过 CSS Variables 实现
- **配色方案**: HSL 变量，主色为 teal 系（`hsl(173 58% 39%)`）
- **组件样式**: shadcn/ui 无样式组件 + Tailwind 类名
- **工具函数**: `cn()` = `clsx` + `tailwind-merge`
- **动画**: `tailwindcss-animate` 插件
- **语义化 tokens**: background, foreground, card, popover, primary, secondary, muted, accent, destructive, border, input, ring, chart-1~5

## 11. Provider 栈（渲染顺序）

```
StrictMode
  └─ BrowserRouter
      └─ HelmetProvider          (SEO / 页面标题)
          └─ QueryClientProvider  (React Query)
              └─ Provider          (Redux Store)
                  └─ PersistGate    (Redux 持久化恢复)
                      └─ Toaster    (Sonner 通知)
                          └─ App    (路由)
```
